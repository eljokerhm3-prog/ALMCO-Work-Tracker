# ALMCO Work Tracker

Flutter + Firebase starter for:
- Multi-site work reports
- Dynamic sites / activities / layers controlled by Admin
- Draft reports
- Completed checkbox
- Multiple photos
- Signature support
- Progress dashboard per site
- Approved Documents per site: PDF + images
- Admin approvals and permissions
- PDF / print report support

## Next setup steps
1. Create Firebase project.
2. Enable Email/Password Authentication.
3. Create Firestore Database.
4. Enable Firebase Storage.
5. Run `flutterfire configure` to generate `firebase_options.dart`.
6. Run `flutter pub get`.
7. Run `flutter run`.
